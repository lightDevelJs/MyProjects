Модуль Magento

Разработать модуль Magento Aidalab/Cdata

1) Модуль добавляет в конфигурацию (System -> Configuration) новый раздел Cdata и единственное поле Token.

2) В модуле создать котроллер, который будет обрабатывать следующие запросы:

http://<website_domain>/clist/
Params:
	GET token – Код для проверки доступа.

Метод выводит список всех зарегистрированных в магазине покупателей (customers) 
в формате JSON (login, firstname, lastname). 
Для исключения несанкционированного доступа используется код авторизации (token).
 Значение token приходящее в запросе сравнивается с сохраненным в конфигурации.
 

Params:
	GET token – Код для проверки доступа.
	GET id – ID покупателя

Метод выводит данные выбранного покупателя по ID.
Для исключения несанкционированного доступа используется код авторизации (token). 
Значение token приходящее в запросе сравнивается с сохраненным в конфигурации.

Требования:

•	Писать чистый, форматированный, понятный код
•	Максимально задействовать встроенные классы и методы Magento и Zend Framework
•	Модуль запаковать в архив, сохраняя правильную структуру папок, 
так чтобы его можно было распаковать в любую копию Magento в корневую директорию и запустить.
